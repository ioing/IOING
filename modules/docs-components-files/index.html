文件资源

组件的文件资源被放置在一个独立的文件夹中

+ components + select-date - index.html - style.css - main.js + static - button.png

在项目目录下的components/目录中建立一个和组件名对应的文件夹，如以 ‘select-date’ 为名来命名一个文件夹时即在组件库中注册了该命名的组件

在该组件文件夹中必须包含一个主文件index.html，主文件的内容是该组件的主体结构，如下示例

示例 line 1 为该组件异步引入了一个渐进式 CSS 样式表（了解 CSS 载入机制）

示例 line 2 数据源 inject 为一个外部模版名，当该值存在时则通过 include 在组件中载入这个模版片段，通过该方法能够把自定义的 CSS 样式或 HTML 片段注入到组件中，从而实现对组件的个性化定制

示例 line 3 为该组件所需 js 核心文件 (了解 JS 载入机制)

示例 line 4 为该组件的主体结构，这是一个时间选择器组件，通过 JS 生成好的视图结构最终被插入到这个结构中
需要注意的是该文件是一个html片段而非是一个包含 head/body 的完整html
